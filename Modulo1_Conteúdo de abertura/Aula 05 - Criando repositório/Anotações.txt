# Criando seu Primeiro Repositório no GitHub Para Compartilhar Seu Progresso :book:

### Requisitos do projeto

- Noções básicas de Git/Github

- [Sugestão de curso gratuito da DIO - Introdução ao Git e Github] (encurtador.com.br/fuxCE)

- Git devidamente instalado (usaremos Gitbash)

- Engajamento e vontade de aprender!

### Desafio

    Reforce seu conhecimento em Git com um desafio de projeto totalmente prático, onde você executará todos os passos para a criação, atualização e sincronização de um repositório no GitHub. Para isso, tenha em mente todas as dicas e direcionamentos apresentados pelo expert nas aulas. Dessa forma, você poderá compartilhar suas anotações e exercícios em seu próprio repositório. Criando assim, o primeiro (de muitos) projetos do seu portfólio :smile:
